# GradeCalc Pro 🎓

A modern, fluid, and lightning-fast Single Page Application (SPA) designed to help students calculate their SGPI and overall percentages instantly. 

### Features ✨
- **Dynamic Subject Tracking**: Add or remove multiple subjects easily.
- **Smart Grading System**: Real-time SGPI and Percentage calculations based on standard grading logic (90%+ = 10 GP, 80%+ = 9 GP, etc.).
- **PDF Report Export**: Generate and download a beautifully formatted PDF report containing your name, roll number, and subject-wise breakdown with a single click.
- **Privacy First**: Fully standalone and runs locally in your browser. No data is stored or sent to any backend servers.
- **Modern UI**: Built with Material 3 aesthetics, sleek glassmorphism, and completely responsive design.

### Tech Stack 💻
- **Frontend**: HTML5, Vanilla JavaScript, React (via CDN)
- **Styling**: Tailwind CSS (via CDN)
- **Exports**: jsPDF
- **Icons**: Lucide Icons
